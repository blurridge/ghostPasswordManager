# Ghost Password Manager by blurridge

## Context
Ghost is a Python-based password manager using tkinter and pandas. The project aims to create a minimal desktop app that could
store, encrypt, and decrypt passwords. 

As of the moment, the project aims to:
1. Implement encryption and decryption for system file that stores the account details.
2. Ensure functionality on both Mac and Windows.
3. Develop a Chromium-based extension version in the future.
